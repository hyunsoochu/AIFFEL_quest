- [ ]  **4. 회고를 잘 작성했나요?**

- [ ]  **5. 코드가 간결하고 효율적인가요?**
    - 파이썬 스타일 가이드 (PEP8) 를 준수하였는지 확인

	    - 주석은 # 뒤에 공백 1칸을 작성해야합니다.
	    - 2줄 이상 작성하는 경우 doc string으로 작성해서 주석을 깔끔하게 작성하는 게 좋을 것 같습니다.
		- 모국어가 영어가 아닌 경우에도 왠만하면 영어로 작성하세요.
